This is a simple and responsive Weather App.
It allows users to check real-time weather information.
Users can search weather by entering a city name.
The app fetches data using a weather API.
It displays temperature, humidity, and wind speed.
It also shows current weather conditions.
The user interface is clean and easy to use.
This project is built using HTML, CSS, and JavaScript.
It helps beginners understand API integration.
Overall, it is a useful and practical web project.
